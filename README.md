# Introduction-to-AI – Nhận diện ký tự biển số xe

Dự án này xây dựng hệ thống nhận diện các ký tự (chữ cái và chữ số) trên biển số xe bằng các phương pháp **Machine Learning / Computer Vision**.

## Dataset

Dataset không được đưa trực tiếp lên GitHub vì dung lượng lớn.

Bạn có thể tải dataset tại đường link sau:

Dataset download:
https://drive.google.com/drive/folders/1y5x14eM_T1ECb9GvqghlI_DSCeM9t4s1?usp=sharing

Sau khi tải về, hãy giải nén và đặt dataset vào đúng thư mục như sau:

```
Introduction-to-AI
│
├ dataset
└ README.md
```

Lưu ý: thư mục dataset phải giữ đúng tên **dataset** để các chương trình trong project có thể đọc dữ liệu đúng cách.
